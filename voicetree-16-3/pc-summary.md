# Module Summary

Module summary for js/pages/ - 10 page components using template string export pattern

## Module Summary

The Page Components module (`js/pages/`) contains all UI page templates for the Brotusphere SPA. It follows a simple export-function pattern where each page exports a function returning an HTML template string. Some pages include initialization functions for client-side dynamic behavior.

**Pages (10 total):**
- `home.js` - Landing page with hero, features, product preview
- `about.js` - About the sour fig mission
- `health.js` - Health benefits information
- `products.js` - Product catalog with cart functionality
- `contact.js` - Contact form
- `science.js` - Scientific research with API integrations
- `sphere.js` - Articles/library carousel
- `profile.js` - User profile page
- `orders.js` - Order history
- `notFound.js` - 404 page
